# Hephaestus.Validation
A simple, lightweight, and extensible validation library for .NET.

Hephaestus.Validation provides a clean and fluent API to build validation rules for your objects, helping you enforce data integrity and separate validation logic from your business models.

## Features
- Fluent API: Define complex validation rules in an easy-to-read, chainable way.

- Lightweight: No heavy dependencies, just focused on validation.

- Extensible: Easily add your own custom validation rules.

- Asynchronous: Built-in support for asynchronous validation rules.
